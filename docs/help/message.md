Message — Telegram message operations.

Routes:
- message/edit — edit message text or keyboard
- message/delete — delete a message
- message/pin — pin or unpin a message
- message/get — retrieve message by ID and version
- message/history — get recent conversation timeline
- message/route — route ambiguous message to session (governor only)

action(type: "message") — lists sub-paths in live API.
