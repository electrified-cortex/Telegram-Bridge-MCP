# v7.7.0 — 2026-05-28

## Added

### Skills plugin

- **`.claude-plugin/`** — TMCP is now a Claude Code plugin source. Add `telegram-mcp`
  to `extraKnownMarketplaces` and enable `telegram-mcp-skills@telegram-mcp` in any
  Telegram-participating pod's `settings.local.json` to auto-provision the
  `telegram-participation` skill fleet-wide.

- **`skills/telegram-participation`** — bundled skill: bootstraps any TMCP-enabled agent
  into an active Telegram session (connection check → session anchor → startup drain →
  post-connect setup → dequeue loop). Previously distributed by copy; now a single
  canonical source via the plugin.

### Help docs

- `docs/help/name-tag.md` — name-tag set/clear reference
- `docs/help/profile/kick-lockout.md` — kick-lockout profile action
- `docs/help/reminder/disable.md`, `enable.md`, `sleep.md` — reminder management
- `docs/help/session/status.md` — session status action
- `docs/help/sub-session.md` — sub-session / child-agent reference

## Changed

- `skills/archive/` relocated to `notes/skills-archive/` — deprecated skills preserved
  in git history, removed from plugin discovery path.
- `package.json` version bumped to 7.7.0.
