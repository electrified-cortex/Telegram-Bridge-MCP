# v7.8.0 — 2026-06-01

## Added

- **Runaway-dequeue rate guard** — `dequeue` now detects and throttles pathological
  rapid-fire dequeue loops.
- **`/hook/animation` route** — server-side hook endpoint with debounce, for animation
  presence driven from hooks.
- **`telegram-participation` skill** — R4 now includes an explicit `profile/load` step and
  a boot animation on connect.

## Changed

- **Public task docs sanitized** — verbatim operator voice transcripts across the `tasks/`
  pipeline distilled into neutral third-person summaries (substance and provenance
  preserved); internal hostnames / deploy paths removed from example text; an absolute
  local path scrubbed from a monitor-path note.
- `package.json` and `.claude-plugin/plugin.json` versions bumped to 7.8.0 (plugin manifest
  re-synced — it had lagged at 7.7.0).
- `packageManager` updated to `pnpm@11.5.0`.
- Task pipeline housekeeping (backlog items moved to `needs-refinement/`, completed items
  archived, duplicates removed).

## Fixed

- **Monitor script paths resolved absolutely** via `__dirname` in service-message
  instructions — fixes the relative-path failure on pods without a local `tools/` dir.
- `monitor.ps1` uses a valid `NotifyFilters::LastWrite` enum member.
- `button-helpers` validates label and `callback_data` length in `buildKeyboardRows()`.
- Disambiguated the `behavior_nudge_typing_rate` service-message copy.

## Removed

- Retired the deprecated `edit_message_text` tool and its test.
